# Photos des responsables

Déposez les portraits ici avec ces noms exacts :

- `directeur.jpg` — Directeur de la Chorale
- `maitre.jpg` — Maître de la Chorale
- `organiste.jpg` — Organiste de la Chorale

Format recommandé : **JPG** ou **WebP**, au moins **800×600 px** (le visage est cadré par `object-[center_20%]` — gardez le visage dans la partie haute).

Si un fichier manque, un placeholder doré avec icône silhouette s'affiche à sa place.

> **Note** : les trois images livrées par défaut sont des portraits libres de droits depuis **Pexels** (licence gratuite sans attribution requise). Remplacez-les par les vraies photos de l'équipe.
