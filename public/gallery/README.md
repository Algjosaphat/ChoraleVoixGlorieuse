# Galerie

Déposez ici vos photos nommées :

- `photo-1.jpg` — grand cliché (mise en avant, 2 rangées)
- `photo-2.jpg`, `photo-3.jpg`, `photo-5.jpg`, `photo-6.jpg` — formats classiques
- `photo-4.jpg` — panorama (occupe 2 colonnes)

Formats recommandés : **JPG** ou **WebP**, 1200–1600 px de large, compressées sous ~300 Ko.

Si un fichier est absent, un placeholder branché avec icône note de musique s'affiche à sa place.
