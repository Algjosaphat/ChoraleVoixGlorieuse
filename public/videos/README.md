# Vidéos du Hero

Déposer ici les fichiers suivants :

- `worship-bg.mp4` — vidéo de fond du hero (muette, en boucle, idéalement 10–30s, 1080p ou 720p).
- `worship-poster.jpg` *(optionnel)* — image de substitution avant le chargement de la vidéo.

Si `worship-bg.mp4` est absent, le hero bascule automatiquement sur un dégradé doré/nuit.

Conseils :
- Format MP4 H.264 + AAC pour la compatibilité la plus large.
- Garder le fichier sous ~5 Mo pour un premier rendu rapide (compresser avec HandBrake ou `ffmpeg`).
- Préférer une vidéo sobre (paysages, bougies, architecture du temple) — le texte se superpose dessus.
